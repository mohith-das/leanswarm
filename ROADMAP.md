# LeanSwarm Roadmap

## V2 priorities

1. **Multi-world simulations**  
   Run branching scenario worlds in parallel so one seed can produce structured what-if trajectories.

2. **Monte Carlo runs**  
   Execute repeated seeded runs with aggregate summaries, spread/variance metrics, and confidence bands.

3. **Intervention testing**  
   Support explicit interventions (policy, messaging, shocks) and compare counterfactual outcomes.

4. **Replay UI**  
   Add a timeline replay surface for ticks, state transitions, and world/agent diffs.

5. **Plugin system**  
   Define plugin hooks for scorers, event injectors, model routers, and post-processing outputs.

6. **Large-scale simulations (>5k agents)**  
   Introduce scheduling/memory upgrades and execution strategies for large populations.

7. **Hybrid local + cloud models**  
   Enable routing policies that combine local inference with cloud escalation by quality/cost/latency targets.
